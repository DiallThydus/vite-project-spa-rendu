# Projet

Librairies du projet :

-   Typescript
-   SweetAlert2
-   Eslint
-   Prettier

## Installation du projet

```
git clone https://github.com/DiallThydus/vite-project-spa-rendu.git
npm install
```

## Lancer le projet en mode développement

```
npm run dev
```

## Générer un build

```
npm run build
```

## Prévisualiser le projet

```
npm run preview
```
